# PDF 导出 A vs B 路线对比测试

目标：用 5 分钟判断浏览器打印路线 A 与服务端 Puppeteer 路线 B 的实际导出质量差异。

## 准备

1. 使用管理员账号登录 `https://gambits.top`。
2. 打开任意一份已有内容的 `/report/{workspaceId}` 页面，建议选择包含中文段落、代码块、表格或较长列表的报告。
3. 确认页面内容已完整加载。

## 操作步骤

1. 点击报告页的 HTML 导出按钮，保存为 `report.html`，本地双击打开并确认样式、中文、代码块可见。
2. 在报告页点击浏览器打印或页面上的“打印/PDF”入口，选择“另存为 PDF”，保存为 `pdf_a_browser_print.pdf`。
3. 回到报告页，点击“下载 PDF（服务端）”或服务端 PDF 导出按钮，等待最长 30 秒，保存为 `pdf_b_server_puppeteer.pdf`。
4. 同时打开两个 PDF，并把原始报告页作为参照。
5. 按下表对四个维度评分，1 分最差，5 分最好。
